## 풀이

* 부모를 타고 가면서 이익 설정

## 문제 유형
* 시뮬레이션
## 시간 및 공간 복잡도
* 시간복잡도: $\mathcal{O}(n)$
* 공간복잡도: $\mathcal{O}(n)$
## 코드
```python
def solution(enroll, referral, seller, amount):
    parent = {}
    profit = dict.fromkeys(enroll, 0)

    # 부모 관계 설정
    for e, r in zip(enroll, referral):
        parent[e] = r

    # 판매 처리
    for s, a in zip(seller, amount):
        cur = s
        money = a * 100

        while cur != '-' and money > 0:
            give = money // 10
            profit[cur] += money - give
            cur = parent[cur]
            money = give

    return list(profit.values())

```
