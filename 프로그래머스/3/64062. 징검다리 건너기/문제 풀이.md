# 문제 풀이

## 해결 과정
1. 접근 방식부터 바로 잡기
2. '몇 명까지 건널 수 있는가'가 아닌 **'N명이 건널 수 있는가'** (이것이 이분 탐색의 핵심 아이디어)
3. 최소 1명부터 최대 N명(바위가 최대로 견딜 수 있는 수)의 중간값을 구함
4. 그 중간값을 이용해 돌마다 건널 수 있는지를 판단
5. 건널 수 없는 돌의 개수와 K와 같아지면 즉, 더이상 건널 수 없다면 right = mid - 1, 건널 수 있다면 left = mid + 1
6. 이런 식으로 left <= right 가 좁혀가면서 최대 가능 수를 구함
