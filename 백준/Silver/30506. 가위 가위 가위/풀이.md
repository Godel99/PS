# 풀이
### 문제 요약
문제에서 요구하는 것은 첫 100판을 가위만 내서 알아낸 승수를 가지고 고정된 값만 내는 머신이 무얼 냈는지를 알아내는 것입니다. 여기서 이용할 방법이 바로 **차분 공격**입니다.
### 차분 공격이란?
**입력을 아주 조금만 바꿨을 때, 출력이 어떻게 달라지는지를 관찰해서 내부의 숨겨진 구조를 알아내는 방법**입니다. 

즉, i번째 마다 100판을 해야 하는데 그 때 i판에 **바위를 한 번 내고 나머지를 전부 가위만** 내서 승수의 차이를 이용해 머신이 무얼 냈는지를 밝혀내는 것입니다.

## 문제 유형
* 차분 공격
## 코드
### Python
```python
import sys
def print(*args, sep=" ", end="\n"):
    sys.stdout.write(sep.join(map(str, args)) + end)
def input():
    return sys.stdin.readline().rstrip()

def main():
    k = int(input())
    ans = ['0'] * 100
    hands = ['2'] * 100
    for i in range(100):
        hands[i] = '0'
        print('?', ''.join(hands))
        sys.stdout.flush()
        hands[i] = '2'
        newk = int(input())
        if k < newk: ans[i] = '2'
        elif k == newk: ans[i] = '0'
        else: ans[i] = '5'
    print('!', ''.join(ans))
    sys.stdout.flush()
if __name__ == '__main__':
    main()
```
### C++
```c++
#include<bits/stdc++.h>
using namespace std;
int ans[101];

int main(){
	int k, newk; cin >> k;
	for(int i = 1; i <= 100; i++){
		cout << "? ";
		for(int j = 1; j < i; j++) cout << 2;
		cout << 0;
		for(int j = i+1; j <= 100; j++) cout << 2;
		cout << endl;
		cin >> newk;
		if(k < newk) ans[i] = 2;
		else if(k == newk) ans[i] = 0;
		else ans[i] = 5;
	}
	cout << "! ";
	for(int i = 1; i <= 100; i++) cout << ans[i];
	cout << endl;
	return 0;
}
```
